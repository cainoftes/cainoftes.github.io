图片格式

七月 15, 2022 发布在 default

参考了 B站 视频 ：

【一口吃胖】图片与格式：计算机通用图像知识与文件格式（BMP、JPG、PNG、GIF、WebP、AVIF、HEIF、JXL）_哔哩哔哩_bilibili

BMP

指位图，即 Bitmap，也被称为 DIB （设备无关的位图）。

微软于 1990年 为 Windows 3.0 而开发。最先由 “画图” 软件支持。

众所周知，微软的产品的兼容性非常强，所以现在还能跑起来的 Winodws 系统都能打开 BMP 格式的文件。

但是 BMP 又是一个很烂的名字，因为 Bitmap 是一系列图片格式的总称（除了 SVG）

不支持压缩：非常大，甚至是 20倍

色深

色深， color depth，又叫 bit depth，因为它的单位是 bit

  • 24位色（1600万色，真彩色，Truecolor）：刚出来的时候被认为是人眼的极限，所有叫“真”，现在有了 10位色 （RGB 各自的值是10位，总共是30位），叫“深”。
  • 256色（8位色）
  • 16色（4位色）
  • 单色（1位色）

像素

像素，Pixel

像素有 屏幕像素（或者叫物理像素）和 图片像素之分。

一般来说，当显示 100% 的时候，一个图片像素就对应了一个 物理像素。当显示 800% 的时候，表明对图片像素颗粒进行了放大（比例是多少？），一个图片像素颗粒对应了 8 * 8 个物理像素颗粒。

当使用其它的看图软件的时候，软件可能会对放大的图片做一些改变（平滑处理）

分辨率

分辨率：Resolution，也叫图片尺寸，举个例子：1920 * 1280 像素

分辨率表明了这一个图片的 width * height，也就是这个图片宽度等于多少像素，高度等于多少像素。

一般来说，宽和高有联系

  • 老式电视 4：3（16：12）
  • 标准宽屏 16：9，用标准宽屏看老式电视，左右会出现黑色的空白。
  • 超宽屏（带鱼屏）：21：9

常见的分辨率

  • 720p（HD，没有记错，在1080p没有出现的时候，720p就是 HD，所以现在一般会避开 HD这个称呼）：1280 * 720
  • 1080p（高清 HD， 全高清 FHD， Full HD）：1920 * 1280
  • 2k：2560 * 1440
  • 4k：3840 * 2160
  • 6k：
  • ……

上面这些是 标准宽屏 16：9

JPEG

有两种后缀：.jpeg 和 .jpg

上面 BMP 中所说的20倍 就是和 JPEG 对比出来的，所以它的压缩率还是很恐怖的。这种格式占据了大半的市场份额。

当然，这个是有缺点的：

  • 有损压缩（.mp3 和 .mp4 都是这种）
  • 不支持半透明

EXIF

用来记录照相的信息，例如在什么位置，用什么相机拍的等

RGB

光的三原色，这是发光体

  • 红黄蓝：颜料三原色，这是吸光体

CMYK

印刷领域：黄，品红，青色，黑色（K，为了和蓝色分开）

Alpha

RGBA：使用了256种透明的RGB

Alpha 通道：半透明通道

PNG

Portable Network Graphics，即：便携网络图像。

多少有点 BYD 的意思了，最初是很流行很骚气的递归缩写：PNG’S Not GIF

无损压缩，强制无损压缩。

一般来说，一张图片经常被编辑，最好使用 Png 而不是 JPEG，不然会很糊。

而且还支持半透明

GIF

  • 无损
  • 半透明：用了一个 bit
  • 动图：存储多张图片和各个图片间的时间间隔

看起来这个东西比前边的都强大，但是他目前最广泛用于斗图，为什么？因为他的各个方面都是半吊子。最高支持 256 色。

APNG

PNG 最初就是为了改进 GIF ，但是对动图并不支持，所以 PNG 官方组开创了 MNG用来支持动图。着MNG 发布之后 FireFox 和 Netscape 在 2002 年加入了对其的支持（IE 在看戏），2003年又删了。后来 Mozzia 公司的两个员工开创了另一个 PNG 的动图格式——APNG，但是被 PNG 官方组拒绝了，后来刚出生的 Chrome 选择了 APNG（IE 在看戏）。于是，APNG 就统一了。

WebP

谷歌在2010年发布的，可以取代 JPG , PNG , GIF

  • 支持无损压缩，而且压缩率比 PNG 高
  • 支持有损压缩，而且压缩率比 JPG 高
  • 支持动画

可惜，Windows 看图软件不支持

缩略图

缩略图：Thumbnail

Mac 的 .DS_Store 和 Windows 的 Thumbs.db。而且你还删不完。

如果同时显示很多图片，不可能直接加载原图的，所以使用了缩略图，200 * 200， 10KB。

AVIF/HEIF

压缩率的天花板，时间代价。

HDR

High Dynamic Range

需要加一个 HDR 通道，使用浮点数表示。但是它的存储空间大（一倍）

JXL

诞生于2021年。

TIFF

打印、扫描、印刷领域最主流的格式

  • 图像模式
  • 图层
  • 多页：Windows 看图软件打开

色彩模型

RGB

CYMK

HSV / HSB ：为了人类创造出来的调色工具。

可以将 HSV 理解为一个 倒置的圆锥体。

HSL：HSV 的一种改进，增加了缺点增加了优点。

色彩空间

sRGB

Adobe-RGB

DCI-P3

PSD

来源于 PS 软件，是一个专用的文件格式。

对比 程序员，可以将以上介绍的图片格式类比为 .exe， 那么设计师的源文件是什么呢？可以是 TIFF ，但是 TIFF 文件会有问题：不同的图片编辑器都可以编辑（不懂），所以各个厂商都有自己的专用格式。

而 PS 开发出来的 PSD 就是用于保存设计信息的一个格式，叫“分层源文件”

RAW

数字底片，没有经过处理。较大的文件体积，专业的软件，专业的处理师。

每一个相机厂商都有自己的格式。

全景图

天空盒

TGA

CG 领域

SVG

分辨率无限高

文件大小又很小

是一种矢量图

使用 XML 格式来存储

只擅长表达几何形状——所以平时的照片几乎不可能使用这种格式

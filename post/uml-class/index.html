UML Class

2022-10-5 发布在 default

更重要的是面向对象建模，而不是UML本身长什么样子

简介

UML：统一建模语言，包含了很多种的图表（8+），主要用于面向对象设计。

UML可以通过很多种软件来绘制，比如visio、draw.io、亿图设计等。

同时还可以用编程语言来实现，代码会转换为UML，这其中最常见的是PlantUML和mermaid（中文：美人鱼）。其中PlantUML类似于编译器，可以通过命令（CMD/Bash）的方式将源文件转换为SVG图片，配合VSCode可以实现分屏实时预览；而Mermaid类似于PlantUML，但是对UML的支持似乎比PlantUML弱一点（UML的图标支持不全面），但是Typora内置了Mermaid，将内嵌代码的格式改为Mermaid会自动显示。

而Mardown领域打遍无敌手的Typora不支持PlantUML，似乎也是一种遗憾。

UML Class，是UML中的一种图，同时也是最常用的图之一（甚至没有之一）。

目录操作

命令名 作用 说明
pwd 打印当前目录 print working directory
hostname 打印主机名
mkdir 创建目录 make directory；如果文件名有空格，则使用双引号，否则会创建多个文件夹；即使中间路径不存在也可以执行——创建中间路径
cd 更改目录 change directory；对于Windows来说，只能在一个盘（专业术语：区）内使用，例如在C盘下，无法cd到D盘，只能先跳转到D盘（命令为：“D:”），然后cd。
ls 列出目录下的文件 list
rmdir 删除目录 remove directory；不会被放到回收站；仅仅针对空的目录
pushd 跳转到path push directory;和popd 配合使用
popd 跳转到上一个pushd的目录 pop directory;和pushd 配合使用

文件操作

命令名 作用 说明
New-Item a.txt -type file 创建文件 需要Powershell；该命令还可以创建目录等；
copy 复制文件 copy
mv 移动文件 move；可以用来重命名
more 查看文件内容
del 删除文件
find 查找文件内容

MySql安装

https://dev.mysql.com/downloads/mysql/

如果C盘空间比较小，可以下载压缩包进行安装到其他盘。以上下载地址提供的是最新版的MySql下载地址，当前是8.0.29版本。

下载上图的第一个。

下载完成之后解压文件。需要注意的是，解压之后的文件相当于已经安装好了，在选择解压目录的时候可以放到自己的常用的安装目录。

解压完成之后在mysql-8.0.29-winx64目录下建立my.ini文件，写入如下内容（8.0.29修改成自己的版本）

[mysqld]
port=3306
basedir=D:\mysql-8.0.29-winx64 
datadir=D:\mysql-8.0.29-winx64\data
max_connections=200
max_connect_errors=10
character-set-server=utf8
default-storage-engine=INNODB
default_authentication_plugin=mysql_native_password
[mysql]
default-character-set=utf8
[client]
port=3306
default-character-set=utf8

将bin目录配置到用户环境变量中（为了在cmd中直接使用bin目录下的命令而不是跳到bin目录下执行命令）。

使用管理员身份打开cmd（防止各种烦人的权限问题）

mysqld --initialize --console

mysqld是MySql的后台程序，也就是mysqld是服务端，mysql是客户端，如果要使用mysql必须开启mysqld。

回车之后记录下临时的密码（后面需要自己输入，不建议复制粘贴），在最后一行的最后，例如

root@localhost: f2QmNsq<uSiM

这是一种常见的格式，@之前是用户名，@之后是主机的位置。例如Cainoftes@126.com就符合这种格式。而 : 之后的字符（不包含空格）是用户名访问该主机所需要使用的密码。

然后使用命令

mysqld --install

这是安装mysql的服务。

再输入

net start mysql

可以看到服务启动成功的提示。

之后就是修改密码的操作。输入如下命令

mysql -u root -p

回车之后可以看到输入密码的提示，输入之后，进入了mysql的界面。

使用如下命令修改

alter user root@localhost identified by 'firefox'

这行命令就是sql语句，将root的密码修改为了firefox.

其实密码是存储在数据库的表中的，而以上两条命令中，第一条使用mysqld为我们建立的临时密码访问数据库，然后对数据库表中的数据进行修改，修改完成之后我们的密码自然改变。

安装完成，使用exit退出sql命令行。

使用

开启MySql服务器
mysqld --console
关闭MySql服务器
mysqladmin -u root -p shutdow

打开cmd，输入以下命令后回车，再输入密码，就成功连接到了数据库。

如果不是连接本地主机，需要使用-h 指定

mysql -u root -p

mysql命令汇总

显示当前有哪些database
show databases;
使用某个数据库
use mysql;
显示mysql数据库下的数据表
show tables;
显示某个数据表下的所有列
show columns from user;
显示某个数据表的所有索引
show index from user;

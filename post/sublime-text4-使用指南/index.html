Sublime Text4 使用指南

九月 30, 2022 发布在 default

安装

官网 https://www.sublimetext.com/

Sublime Text是一个文本编辑器（不能称为IDE），但是这种文本编辑器又和Windows自带的文本编辑器不一样（自带的弱爆了），所以常常会被忽略了其本质，称其为IDE，这是不严格的。

Sublime Text是一个闭源收费的软件（……），但是可以一直免费用（可能会有弹窗/在左上角标注UNREGISTERED）

Sublime Text 主打的功能：打开文件的速度非常快。

配置环境变量

可以将Sublime Text的安装目录配置到环境变量中，之后可以使用 subl命令在CMD中打开文件，文件夹。

安装Package Control

官网 https://packagecontrol.io/

众所周知，没有了插件的编辑器没办法成为一个 “IDE”，所以对于插件的管理就编程了重中之重。

Package Control 本身就是一个插件，但是它的特殊之处在于：它是管理插件的插件，所以将Package Control 的安装放到了Sublime Text的安装目录下，本身就意味着，没有Package Control 的 Sublime Text 毫无灵魂。

安装过程如下：

使用快捷键 Ctrl + Shift + P，调出来命令面板，在其中输入 install package control，回车之后自动安装。

如何使用Package Control 见下文 插件 部分。

快捷键

通用

  • Alt : 调出菜单
  • Ctrl + Shift + P : 调出命令面板
  • Ctrl + ` ：调出控制台

编辑

  • Ctrl + Enter ：在当前行下面新增一行（不管光标在当前行处于何种位置），然后跳转到新行
  • Ctrl + Shift + Enter：类似上条，只不过是在上面增加一行
  • Ctrl +↑ ： 移动显示区域
  • Ctrl + Shift + ↑ : 移动当前行

选择

  • Ctrl + D ：选择当前光标所在的词并高亮该词出现的位置

    • Ctrl + D : 多重选择（选择下一个和选中词相同的词语）
    • Ctrl + K : 跳过当前选中的这个
    • Ctrl + U ：回退

  • Ctrl + Shift + L：将当前选中的区域打散（分词，每个词同时编辑）

  • Ctrl + M： 在起始括号和结尾括号之间跳转

  • Ctrl + Shift + M：选中括号间的所有内容

  • Ctrl + L：选择整行

  • Ctrl + Shift + L：同时编辑选中的多行的行尾

  • Ctrl + Shift + [ ：折叠代码

  • Ctrl + Alt + ↑ ：多行编辑

  • Ctrl + K + U：转换为大写

查找替换

  • F3：跳转到当前关键字的下一个位置
  • Alt + F3：选中当前关键字出现的所有位置

跳转

  • Ctrl + P：跳转到指定文件

    • @：跳转到指定的符号
    • #：关键字跳转
    • : :行号跳转

窗口

  • Ctrl + Shift + T：恢复刚刚关闭的标签

屏幕

  • F11：切换普通全屏
  • Shift + F11：切换无干扰全屏
  • Alt + Shift + 1：取消分屏
  • Alt + Shift + 2：进行左右分屏
  • Alt + Shift + 8：进行上下分屏
  • Alt + Shift + 5：进行上下左右分屏

插件

概述

推荐使用Package Control 安装和管理插件。

插件的本地存放位置为 “.\Packages\”

移除插件

启动命令面板（快捷键：Ctrl + Shift + P），输入

package control: Remove Package

实际上，命令面板的命令搜索机制还是很强大的，只要输入每个词的首字母，基本能跳出来，例如说上面的命令可以使用 PCRP或者RP，就能搜索出来。

之后会出现所有已经安装过的插件，通过上下键来选中想要删除的插件，回车，会自动删除。

安装插件

启动命令面板，输入

package control: install package

当然，也可以输入 PCIP 或者IP，应该能搜索出来。

需要注意的是，安装完插件之后关闭，然后再次打开（这就是重启，但是Sublime Text 速度太快了，有种不真实的感觉）。有些插件如果不重启可能不会生效。

推荐插件目录

  • SideBarMenuAdvanced

    让侧边栏（显示出来的文件）有更多的功能，

  • Alignment

    让多行程序的 = 可以自动对其，其快捷键为 Ctrl + Alt + A

  • AutoFileName

    本地文件名的提醒

  • TrailingSpaces

    去除多余的空格；

    安装之后重启，就可以高亮显示多余的空格

元素显示模式

块元素

  1. 独占一行
  2. 高度、宽度、内外边距可以自定义
  3. 默认宽度是父容器的100%
  4. 是一个容器及盒子，里边可以放行内元素或块级元素
  5. 文字类的元素内不能使用块级元素，例如<h1-h6>，<p>内不能放置<div>

行内元素/内联元素

  1. 宽度和高度直接设置无效
  2. 默认宽度是它本身的宽度
  3. 行内元素只能容纳行内元素或文本
  4. 链接里边不能再放链接
  5. 链接里边可以放块级元素：什么是链接：<a>

行内块元素

img, input, td
  1. 一行可以有多个，但是中间会有空白空隙
  2. 默认宽度是它本身的宽度
  3. 高度、行高、外边距以及内边距都可以控制

设置背景

background-repeat: repeat|no-repeat|repeat-x|repeat-y;
background-position:x y;
background-attachment:scroll|fixed
  1. x: 可以是left|center|right，y：可以是bottom|center|top
  2. 如果x|y是方位词，而另一个省略了，默认为center。

优先级

从高到低：!important > 行内样式 > ID选择器 > 类选择器/伪类选择器 > 元素选择器 > 继承/*

需要注意的是：如果是继承的样式，只要别的地方有修改就不起作用了。

这样就解释了为什么修改<p>的时候<a>并不会改变
